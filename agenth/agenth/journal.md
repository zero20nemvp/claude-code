# AgentH Journal

Log your observations here using `/journal <entry>`.

The system analyzes patterns and suggests automation opportunities after 3+ similar mentions.

**What to log:**
- Task reality vs expectations
- Estimation accuracy observations
- Friction points and repetitive work
- Tool wishes ("wish I had...", "should automate...")
- Anxiety about specific goals

**The system will:**
- Detect patterns automatically
- Suggest building tools to automate repetitive work
- Adapt task generation based on your observations
- Recalibrate effort estimates using real data

Just observe and log. The system adapts automatically.
